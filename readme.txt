Visa Requirements Widget Plugin

== Description ==

This plugin adds a powerful widget to your WordPress site, allowing users to check visa requirements for 199 different passports and 209 destinations. It delivers real-time updates on visa policies, ensuring your audience has access to the latest travel information.

== External Services ==

This plugin relies on the Travel Buddy API to provide real-time visa requirements and travel information. When users check visa requirements using the widget, their request is sent to the Travel Buddy API, which processes the request and returns the necessary information.

- **Service Name:** Travel Buddy API
- **Service URL:** [https://rapidapi.com/TravelBuddyAI/api/visa-requirement](https://rapidapi.com/TravelBuddyAI/api/visa-requirement)
- **API key:** API key can be requested [here](https://rapidapi.com/TravelBuddyAI/api/visa-requirement)

== Installation ==

= Automatic Installation =

1. Go to Plugins -> Add New in the WordPress admin panel.
2. Type "Travel Buddy" in the search bar.
3. Click the 'Install' button found under the Travel Buddy plugin.
4. After installation, activate the plugin by clicking on 'Activate'.

= Manual Installation =

1. Login to the WordPress admin panel and select Plugins -> Add New.
2. Choose the 'Upload' option.
3. Click 'Choose File' and select the travel-buddy.zip file you have downloaded.
4. Click 'Install Now'.
5. To activate, find the Travel Buddy under Plugins -> Installed Plugins and click 'Activate'.
